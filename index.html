About the Project

This repository aims to collect and structure information related to faith-based relationships, with an initial focus on Christian doctrine, for the purpose of training AI models.

Goal

To create a comprehensive, structured, and community-driven dataset that can power AI applications for relationship coaching, counseling, and guidance from a faith perspective.

Inspiration

The design draws inspiration from existing structured data repositories, particularly BradyStephenson/bible-data, for its methodical approach to linking entities (people, places, events) and referencing sources within biblical texts.

Data Schema

Data is stored in JSON format, with each file typically containing an array of data entries. Each entry adheres to the structure defined in docs/data_schema.json.

Entry

entry_id string Unique identifier
topic array Primary topics
sub_topic string More specific sub-topic
faith_perspective string Primary perspective
source_type string Type of source material
source_reference object Reference to original source
content_summary string Brief summary
content_detailed string Detailed explanation
keywords array Relevant keywords
related_scripture array Related scripture references
related_entries array Related concepts
qa_pairs array Q&A pairs for AI training
instruction_response array Examples for instruction models
metadata object Entry metadata
Example Schema (Simplified) 
{
  "entry_id": "SCRPT_PROV_15_1_GENTLE_ANSWER",
  "topic": ["Conflict Resolution", "Communication"],
  "sub_topic": "Responding Gently",
  "faith_perspective": "Christian",
  "source_type": "Scripture",
  "source_reference": {
    "text": "Proverbs 15:1 (NIV)",
    "url": "https://www.biblegateway.com/passage/?search=Proverbs+15%3A1&version=NIV"
  },
  "content_summary": "Contrasts the effect of gentle versus harsh communication in conflict.",
  "content_detailed": "A gentle answer defuses anger (\"turns away wrath\"), while a harsh word escalates it (\"stirs up anger\").",
  "keywords": ["communication", "conflict", "gentleness", "anger"],
  "related_scripture": [
    { "text": "Ephesians 4:29" }
  ],
  "qa_pairs": [
    {
      "question": "How should I respond when my spouse is angry?",
      "answer": "Proverbs 15:1 teaches that a gentle answer turns away wrath. Responding with gentleness rather than harshness can help de-escalate conflict."
    }
  ],
  "metadata": {
    "creation_date": "2023-05-15",
    "last_modified_date": "2023-05-15",
    "contributor": "Initial Seed",
    "version": 1,
    "moderation_status": "Approved"
  }
}

Example Entries

The dataset contains various types of entries from different sources, each following the standardized schema.

Scripture-Based Entry

SCRPT_PROV_15_1_GENTLE_ANSWER
Source: Proverbs 15:1 (NIV)
Topics:
Conflict Resolution Communication Anger Management
Summary: Contrasts the effect of gentle versus harsh communication in conflict: a gentle answer defuses anger ("turns away wrath"), while a harsh word escalates it ("stirs up anger").
Q&A Example:
How should I respond when my spouse is angry?
Proverbs 15:1 teaches that a gentle answer turns away wrath. Responding with gentleness rather than harshness can help de-escalate conflict.

Theological Entry

THEO_MARRIAGE_COVENANT
Source: Theological Commentary on Marriage Covenant
Topics:
Marriage Covenant Commitment
Summary: Explores the theological understanding of marriage as a covenant relationship that reflects God's covenant with His people.
Q&A Example:
What does it mean that marriage is a covenant?
A covenant is more than a contract; it's a sacred bond. In Christian theology, marriage as a covenant means it's a binding relationship established before God, reflecting His faithful commitment to His people.

Counseling Entry

COUNSEL_ACTIVE_LISTENING
Source: Christian Counseling Principles
Topics:
Communication Listening Empathy
Summary: Outlines the importance and techniques of active listening in Christian relationships, emphasizing presence, attention, and empathetic response.
Q&A Example:
How can I be a better listener to my partner?
Active listening involves giving your full attention, maintaining eye contact, avoiding interruptions, reflecting back what you hear, and responding with empathy. This approach honors your partner and follows the biblical principle of being "quick to listen, slow to speak" (James 1:19).

Practical Advice Entry

PRACT_CONFLICT_RESOLUTION
Source: Faith-Based Relationship Guide
Topics:
Conflict Resolution Forgiveness Reconciliation
Summary: Provides practical steps for resolving conflicts in relationships from a faith perspective, including prayer, honest communication, and seeking reconciliation.
Q&A Example:
What steps should we take when we have a serious disagreement?
First, pray together for wisdom and understanding. Second, each person should express their perspective using "I" statements while the other listens actively. Third, identify the core issue beneath the disagreement. Fourth, brainstorm solutions together. Finally, commit to a resolution and follow through, extending forgiveness as Christ forgave us (Colossians 3:13).

Applications & Use Cases

This dataset can be used for various purposes in AI, research, content creation, and personal study.

AI Training

Fine-tuning or providing context (via RAG) for Large Language Models (LLMs) to create AI-powered relationship coaches, counselors, or chatbots grounded in Christian principles.

Example:

A faith-based relationship counseling chatbot that provides biblically-informed advice on communication, conflict resolution, and marriage topics.

Research

Analyzing patterns in faith-based relationship advice, scriptural interpretations, or counseling themes.

Example:

Academic study on how biblical principles are applied to modern relationship challenges across different theological traditions.

Content Creation

Serving as a structured knowledge base for articles, study guides, or educational materials on Christian relationships.

Example:

Marriage preparation course materials that systematically cover biblical principles for healthy relationships.

Personal Study

Exploring interconnected concepts and scriptures related to relationship topics.

Example:

Individual or couple's devotional study on biblical teachings about love, respect, and commitment in relationships.
